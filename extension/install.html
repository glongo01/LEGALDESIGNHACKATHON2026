LEXIA

EU AI Act Rights Checker — installation guide

1

Start the API server

Open a terminal in the LEXIA-Hackathon/ folder and run:

python backend/api/app.py

Keep the terminal open. The server runs on localhost:5050.

2

Open Chrome / Opera extensions

Open chrome://extensions

Or paste chrome://extensions in the address bar.

3

Enable Developer Mode

Toggle Developer mode in the top-right corner of the extensions page.

4

Load the extension

Click Load unpacked and select the extension/ folder inside LEXIA-Hackathon/.

5

Visit a supported site

Go to chatgpt.com, claude.ai, x.com, klarna.com, or facebook.com.

LEXIA shows a notification automatically on your first visit of the day, and displays the score badge on the extension icon.

Supported sites: chatgpt.com · openai.com · claude.ai · anthropic.com · x.com · twitter.com · klarna.com · facebook.com

The API server must be running for results to appear. If the banner does not appear, click the LEXIA icon in the toolbar.
